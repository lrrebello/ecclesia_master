Modelo 25 - Donativos Recebidos

Declaração anual de donativos para a Autoridade Tributária

O que é o Modelo 25?

O Modelo 25 é uma declaração fiscal obrigatória em Portugal para entidades que recebem donativos, incluindo igrejas. A declaração deve ser enviada eletronicamente ao Portal das Finanças até o final de fevereiro de cada ano, reportando os donativos recebidos no ano anterior.

Benefício para os doadores: Os contribuintes podem deduzir 25% do valor dos donativos na coleta do IRS (limite de 15% da coleta total), conforme o artigo 63.º do Estatuto dos Benefícios Fiscais.

Gerar Relatório
Normalmente, o Modelo 25 é gerado em fevereiro para o ano anterior.
Como Usar
  1. Selecione o ano fiscal desejado
  2. Clique em "Pré-visualizar Dados" para revisar
  3. Verifique se todos os doadores têm NIF válido
  4. Gere o relatório em Excel
  5. Importe os dados no software oficial do Portal das Finanças
  6. Emita comprovativos individuais para os doadores
Requisitos Importantes
Dados da Igreja
  • ID Fiscal (NIF/CPF/CNPJ) da igreja deve estar cadastrado
  • Endereço completo deve estar atualizado
Dados dos Membros
  • Apenas doadores com ID Fiscal válido para o país serão incluídos
  • Donativos devem estar vinculados ao membro
  • Recomenda-se atualizar os IDs Fiscais antes de gerar o relatório
